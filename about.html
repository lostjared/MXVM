Project Overview

MXVM is a comprehensive virtual machine implementation written in C++ that serves as a learning tool for understanding compiler design, virtual machine architecture, and low-level programming concepts.

Key Learning Concepts

  • Lexical Analysis: Tokenization and regular expression matching
  • Syntax Analysis: Recursive descent parsing and AST construction
  • Semantic Analysis: Type checking and symbol resolution
  • Code Generation: Assembly output and linking
  • Runtime Systems: Stack management and garbage collection
  • Dynamic Loading: Plugin architecture with shared libraries

Architecture Components

Scanner/Lexer

Tokenizes MXVM source code, handles comments, identifiers, numbers, and operators with lookahead capabilities.

Parser

Builds Abstract Syntax Tree from tokens, supports program/object parsing with section handling.

Validator

Validates syntax and semantics, checks instruction validity and variable declarations.

Virtual Machine

Stack-based execution engine with memory management and dynamic module loading.

Execution Modes


Interpreter Mode:
- Direct execution of MXVM bytecode
- Real-time variable inspection
- Interactive debugging capabilities

Compiler Mode:
- Generates x86_64 assembly code
- Links with system libraries
- Produces standalone executables

Program Structure

    section data {
        int counter = 0
        string message = "Hello World: %d\n"
    }
    
    section code {
    start:
            mov counter, 1
            print message
            done
    }
}

Building MXVM

Prerequisites

  • C++20-compatible compiler (gcc or clang)
  • CMake ‚â• 3.16
  • System libraries: dl, pthread

Build Instructions


# Clone and build
git clone https://github.com/lostjared/MXVM.git
cd MXVM
mkdir build && cd build
cmake ..
make
sudo make install


# Run example
mxvmc input.mxvm --path /usr/local/lib --object-path .
# Compile example
mxvmc input.mxvm --path /usr/local/lib --object-path . --action translate

Usage Examples


mxvmc input.mxvm --path /usr/local/lib --object-path . --action translate --html 
mxvmc input.mxvm --path /usr/local/lib --object-path . --debug  -it
